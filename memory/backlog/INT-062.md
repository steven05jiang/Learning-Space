# INT-062: Disconnect clears token_scopes and deletes bookmarks

**Status:** ⏳ Pending
**Feedback:** N/A
**Priority:** MEDIUM | **Effort:** S
**Depends on:** DEV-082
**Design spec:** `docs/design-twitter-integration.md` §6.3

## Description

Integration test verifying that `DELETE /integrations/twitter/disconnect` correctly clears `token_scopes`, deletes all `twitter_bookmarks` rows for the user, and (when the account is not a login account) removes the `user_accounts` row entirely.

## Acceptance Criteria

- [ ] After disconnect: `user_accounts.token_scopes` is NULL for the Twitter account
- [ ] All `twitter_bookmarks` rows for this user are deleted
- [ ] If Twitter account was not the user's login account: `user_accounts` row deleted entirely
- [ ] If Twitter is the user's only account: row kept but scopes cleared
- [ ] `GET /integrations/twitter/status` returns `connected=false` after disconnect
- [ ] Returns 204 No Content
