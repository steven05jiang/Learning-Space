# DEV-065: Fix graph node popup overflow

**Status:** ⏳ Pending
**Feedback:** FB-005
**Priority:** MEDIUM | **Effort:** XS
**Depends on:** DEV-052 ✅

## Description

The graph node popup that appears on node click overflows its container because the summary text makes it too long. Constrain the popup to a fixed max-width/height, remove the summary field entirely, truncate the title with ellipsis, and show only: title, URL (as a clickable link), and tags.

## Acceptance Criteria

- [ ] Popup max-width: 320px; max-height: 240px; overflow: hidden
- [ ] Summary field not rendered in the popup
- [ ] Resource title truncates at 1 line with ellipsis (`text-overflow: ellipsis`)
- [ ] URL rendered as a clickable link (`<a target="_blank">`)
- [ ] Tags rendered as compact chips; no horizontal overflow (wrap with hidden overflow)
- [ ] Popup remains readable and non-overflowing on both short and long resource titles
