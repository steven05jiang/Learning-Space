## Description

Keep the chat panel openable via the Sparkles toggle but disable the ability to send messages. The message input and send button should be grayed out and non-interactive. On open, inject a first bot message indicating the feature is coming soon, so users understand the intent of the panel.

## Acceptance Criteria

- [ ] Chat panel still opens/closes via Sparkles toggle
- [ ] Message input field is disabled (grayed out, `disabled` attribute)
- [ ] Send button is disabled (grayed out, non-clickable)
- [ ] On panel open, an initial bot message appears: "AI chat is coming soon — stay tuned! 🚀"
- [ ] The initial message renders in the bot message style (left-aligned, bot avatar/indicator)
- [ ] `make web-lint` passes
