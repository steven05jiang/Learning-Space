## Description

Extend the `process_resource` worker pipeline to generate an embedding vector after the LLM step and upsert it into `resource_embeddings`. The text to embed is built by `build_embedding_text()` (title + summary + tags + top_level_categories). On re-processing, the existing embedding row is updated. On deletion, the FK CASCADE handles cleanup automatically. Adds `EMBEDDING_MODEL` and `EMBEDDING_API_KEY` env vars.
