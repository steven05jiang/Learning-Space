## Description

Provision the three managed data services needed for production: Supabase (PostgreSQL), Neo4j AuraDB Free (graph DB), and Upstash (Redis for ARQ queue). Record all credentials and connection strings in `memory/infra.md` (the SSOT for infrastructure).

## Acceptance Criteria

- [ ] Supabase project created; PostgreSQL connection string (pooled + direct) recorded in `memory/infra.md`
- [ ] Neo4j AuraDB Free instance created; bolt URI + username + password recorded in `memory/infra.md`
- [ ] Upstash Redis instance created; Redis URL (redis://) recorded in `memory/infra.md`
- [ ] All credentials stored as secrets (not committed to repo)
- [ ] `memory/infra.md` updated with service URLs, regions, tier info
