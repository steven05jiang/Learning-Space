## Description

Integration test verifying that `POST /integrations/twitter/authorize` returns a valid `redirect_url`, and that the `/auth/callback` with `state=twitter_bookmark_auth:<user_id>` correctly updates `user_accounts.token_scopes` with the full scope string.

## Acceptance Criteria

- [ ] `POST /integrations/twitter/authorize` returns `{ redirect_url }` containing all 4 scopes in the URL
- [ ] Simulated `/auth/callback?state=twitter_bookmark_auth:<user_id>` updates `token_scopes` on the correct `user_accounts` row
- [ ] `has_scope(account, 'bookmark.read')` returns true after successful callback
