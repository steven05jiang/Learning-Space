## Requirements

Configure the PostgreSQL data layer for Learning Space. This includes:

- Alembic migration setup wired to the FastAPI app
- Users table (id, email, display_name, avatar_url, created_at, updated_at)
- Accounts table (id, user_id FK, provider, provider_account_id, access_token, refresh_token, created_at) — supports multi-provider OAuth linking
- Resources table (id, user_id FK, url, raw_text, title, summary, tags JSONB, status enum [PENDING/PROCESSING/READY/FAILED], created_at, updated_at)
- SQLAlchemy async models matching the schema
- `alembic upgrade head` must apply cleanly to a fresh PostgreSQL instance
