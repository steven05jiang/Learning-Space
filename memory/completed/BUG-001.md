## Problem

`python-jose` pulls in `ecdsa` (CVE-2024-23342, no fix available). `authlib` is already a dependency and supports JWT natively.

## Implementation Summary

Migrated `apps/api/core/jwt.py` from `python-jose` to `authlib.jose`. Removed `python-jose[cryptography]` from `pyproject.toml`. JWT encode/decode rewritten using authlib header-dict pattern. CVE-2024-23342 (ecdsa transitive vulnerability) eliminated. All 91 unit tests pass.
