## Requirements

Extend the existing `ResourceSearchService` (from DEV-073) with vector search and hybrid RRF merge. A `SEARCH_MODE` env var selects the retrieval strategy. The HTTP endpoint and agent tool contracts are unchanged — only service internals change.

### New env var

`SEARCH_MODE` — `keyword` (default, Phase 1 behavior) | `hybrid` (Phase 2)

### New methods on ResourceSearchService

**`_vector_search()`** — cosine similarity search using pgvector:

```sql
SELECT r.*, 1 - (re.embedding <=> :query_embedding::vector) AS similarity
FROM resources r
JOIN resource_embeddings re ON re.resource_id = r.id
WHERE r.owner_id = :owner_id
  AND r.status = 'READY'
  AND (:tag IS NULL OR r.tags ? :tag)
ORDER BY re.embedding <=> :query_embedding::vector
LIMIT :limit;
```

**`_hybrid_search()`** — RRF merge of full-text + vector results (k=60):

```python
async def _hybrid_search(self, session, owner_id, query, tag, limit, offset):
    k = 60
    candidates = limit * 2
    full_text, _ = await self._full_text_search(session, owner_id, query, tag, candidates, 0)
    query_embedding = await self._embed(query)   # calls SiliconFlow embedding API
    vector, _ = await self._vector_search(session, owner_id, query_embedding, tag, candidates)

    scores: dict = {}
    all_items: dict = {}
    for rank, item in enumerate(full_text):
        scores[item.id] = scores.get(item.id, 0.0) + 1.0 / (k + rank + 1)
        all_items[item.id] = item
    for rank, item in enumerate(vector):
        scores[item.id] = scores.get(item.id, 0.0) + 1.0 / (k + rank + 1)
        all_items[item.id] = item

    sorted_ids = sorted(scores, key=lambda i: scores[i], reverse=True)
    total = len(sorted_ids)
    page = sorted_ids[offset:offset + limit]
    result = []
    for id_ in page:
        item = all_items[id_]
        item.rank = scores[id_]
        result.append(item)
    return result, total
```

**Updated `search()` dispatcher:**

```python
async def search(self, owner_id, query, tag, limit, offset) -> SearchResult:
    async with self._session_factory() as session:
        if settings.SEARCH_MODE == "hybrid":
            items, total = await self._hybrid_search(session, owner_id, query, tag, limit, offset)
        else:
            items, total = await self._full_text_search(session, owner_id, query, tag, limit, offset)
    return SearchResult(resources=items, total=total)
```

### `_embed()` helper

Calls SiliconFlow embedding API (same client from DEV-079) with the search query string. Returns `list[float]` of length 2048.

### Graceful degradation

If `SEARCH_MODE=hybrid` but embedding call fails (API error, timeout), fall back to full-text search and log a warning. Do not surface error to user.
