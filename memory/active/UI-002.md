## Summary

Bug fixes and visual polish for the AI chat panel and knowledge graph.

## Scope

### Chat Panel Fixes
- Fixed layout bug where input box was pushed off-screen after multiple messages
  - Added `min-h-0` to `ScrollArea` to constrain flex growth
- Fixed auto-scroll not working with Radix UI `ScrollArea`
  - Replaced `scrollRef.scrollTop` with a sentinel `div` + `scrollIntoView`
- Fixed knowledge graph not re-centering when chat panel opens/closes
  - Replaced `window resize` listener with `ResizeObserver` on the container
  - Calls `graphRef.centerAt(0, 0, 300)` on resize

### Knowledge Graph Visual Enhancements
- Edges now stop at node boundary instead of passing through center
  - Implemented custom `linkCanvasObject` with `'replace'` mode
  - Offsets line start/end by each node's radius
- Edge thickness tuned: normal `1`, highlighted `2`
- Node fill opacity reduced to `0.45` (dimmed: `0.08`) for a more transparent look
