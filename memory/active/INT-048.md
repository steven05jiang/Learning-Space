## Requirements

Write frontend integration tests (Jest + MSW) for the chat UI. Both INT tasks are grouped into a single file.

### Framework

Follow the exact patterns in `apps/web/__tests__/integration/int_chat_ui.test.tsx` (check if it exists) or other frontend INT tests in `apps/web/__tests__/integration/`. Use MSW to mock API calls.

### Test: INT-048 — User opens the chat interface

```
Scenario: User clicks the chat toggle (Sparkles button) — chat panel slides open

Expected:
  - Chat panel becomes visible (translate-x-0 applied)
  - Input field is enabled (not disabled)
  - Send button is enabled
  - Initial state: empty messages list (no "coming soon" text)
```

### Test: INT-049 — User sends a message and receives a response

```
Scenario: User types a message and submits

Setup MSW handler:
  POST /chat → { conversation_id: "test-uuid", message_id: "msg-uuid", response: "Here are your Python resources.", role: "assistant" }

Steps:
  1. Open chat panel
  2. Type "What Python resources do I have?" in input
  3. Click Send (or press Enter)
  4. Verify loading indicator appears briefly
  5. Verify assistant response "Here are your Python resources." appears in chat
  6. Verify input is cleared after send
```
