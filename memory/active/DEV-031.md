## Requirements

Write unit tests for the Knowledge Graph API endpoints. Coverage targets:

- GET /graph — returns nodes + edges for the user's personal graph (happy path + empty graph)
- POST /graph/expand — expands a node and returns neighboring nodes + edges
- GET /graph/nodes/{node_id}/resources — returns resources associated with a tag node
- Auth: unauthenticated requests return 401
- Error cases: node not found returns 404

All Neo4j calls must be mocked. Tests must work offline (no real DB).

Reference: `apps/api/routers/graph.py` and `apps/api/services/graph_service.py` for the implementation.
