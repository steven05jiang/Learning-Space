## Requirements

Abstract the LLM client behind a provider interface so a single `LLM_PROVIDER` env var
selects between Anthropic, Groq, SiliconFlow, and Fireworks AI.

### Why
Production cost: Anthropic claude-haiku is ~$0.25/MTok input. Groq/SiliconFlow/Fireworks
offer equivalent open-source models (Llama 3, Gemma) at ~$0.07–0.10/MTok or free tier.
This lets the app run cheaply in production without code changes.

### Current state
- `apps/api/core/config.py`: `anthropic_api_key`, `anthropic_model`
- LLM is used in: `apps/api/services/llm.py` (or similar) for resource summarization,
  and in `apps/api/services/agent.py` (LangGraph) for chat

### Config changes: `apps/api/core/config.py`

Add:
```python
# LLM provider: "anthropic" | "groq" | "siliconflow" | "fireworks"
llm_provider: str = "anthropic"

# Provider API keys (only the one matching llm_provider needs to be set)
groq_api_key: str = ""
siliconflow_api_key: str = ""
fireworks_api_key: str = ""

# Model overrides per provider (sensible defaults below)
groq_model: str = "llama-3.1-8b-instant"
siliconflow_model: str = "Qwen/Qwen2.5-7B-Instruct"
fireworks_model: str = "accounts/fireworks/models/llama-v3p1-8b-instruct"
```

### LLM service abstraction

Create or update `apps/api/services/llm_client.py` (or wherever the LLM client is initialized):

```python
def get_llm_client():
    """Return a LangChain-compatible chat model based on LLM_PROVIDER setting."""
    provider = settings.llm_provider.lower()
    if provider == "anthropic":
        from langchain_anthropic import ChatAnthropic
        return ChatAnthropic(
            model=settings.anthropic_model,
            api_key=settings.anthropic_api_key,
        )
    elif provider == "groq":
        from langchain_groq import ChatGroq
        return ChatGroq(model=settings.groq_model, api_key=settings.groq_api_key)
    elif provider == "siliconflow":
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(
            model=settings.siliconflow_model,
            api_key=settings.siliconflow_api_key,
            base_url="https://api.siliconflow.cn/v1",
        )
    elif provider == "fireworks":
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(
            model=settings.fireworks_model,
            api_key=settings.fireworks_api_key,
            base_url="https://api.fireworks.ai/inference/v1",
        )
    else:
        raise ValueError(f"Unknown LLM_PROVIDER: {provider}")
```

### Usage sites to update

Find all places that directly instantiate `ChatAnthropic` or use `anthropic_api_key` and
replace with `get_llm_client()`. This typically affects:
- The resource summarization service (LLM call to generate title/summary/tags)
- The LangGraph agent initialization (chat)
