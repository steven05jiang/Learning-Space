**Type:** Backend
**BDD Reference:** Scenario: User views resources for a graph node
**Description:** Query PostgreSQL for resources where `tags` contains the given tag name (node_id maps to a tag label) and `owner_id` matches the authenticated user. Return a list of resources.

**Acceptance Criteria:**
- [ ] Returns resources matching the tag for the current user (owner_id scoped)
- [ ] Uses GIN index on `tags` JSONB column for efficient query
- [ ] Response includes `id`, `title`, `summary`, `original_content`, `content_type`, `status`
- [ ] 404 if no matching resources (or empty list — acceptable)
- [ ] 401 if unauthenticated
